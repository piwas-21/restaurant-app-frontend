# Craft template — Stitch design screenshots

Generated Google Stitch design screenshots for the craft tenant template, produced
from [`../craft-stitch-prompts.md`](../craft-stitch-prompts.md).

**Workflow:** paste a prompt from `craft-stitch-prompts.md` into
[stitch.withgoogle.com](https://stitch.withgoogle.com), then save the generated
design here — one subfolder per prompt (e.g. `02-hero/`), keeping the light + dark
pair — and tick that row in the doc's Progress Tracker. Implement each surface
against its saved design via the per-surface slot mechanism (`src/templates/craft/`).
